Créer un zip sans les fichiers DS_Store sous MacOS X

Si comme moi vous en avez marre de trouver des fichiers DS_Store dans les zip que vous créez, voici une petite commande bien pratique qui vous permettra de créer une archive propre (sans les fichiers MAC) :


	zip -r archive.zip dossier_source -x "*.DS_Store"
